# Brazilian E-commerce Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes Brazilian e-commerce sales data using Python and Power BI to identify revenue trends, top-selling products, category performance, and regional sales insights.

---

## 🎯 Objectives
- Analyze monthly revenue trends
- Identify top-performing products
- Analyze category-wise sales
- Evaluate regional performance
- Generate business insights and recommendations

---

## 🛠 Tools & Technologies
- Python (Pandas, Matplotlib)
- Google Colab
- Power BI
- CSV Dataset

---

## 📊 Dashboard Features
- KPI Cards
  - Total Revenue
  - Total Orders
  - Total Customers

- Revenue Trend Analysis
- Product Performance Analysis
- Category-wise Revenue
- Regional Sales Analysis
- Interactive Filters

---

## 📈 Key Insights
- Certain product categories dominate total sales revenue.
- Revenue shows seasonal fluctuations across months.
- Top products contribute significantly to overall business revenue.
- Specific states generate higher sales performance.

---

## 🎯 Business Recommendations
- Focus marketing on high-performing regions.
- Increase inventory during peak sales periods.
- Promote best-selling products strategically.
- Improve performance in low-revenue categories.

---

## 📂 Project Files
- `Sales_Analysis.ipynb` → Python analysis notebook
- `Brazilian_Ecommerce_Dashboard.pbix` → Power BI dashboard
